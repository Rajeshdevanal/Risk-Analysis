# 🏦 Bank Risk Analysis | Data Analytics Project

## 📌 Project Overview

The **Bank Risk Analysis Project** analyzes banking customer data to understand financial behavior, loan distribution, deposit trends, and credit usage.

This project demonstrates an **end-to-end data analytics workflow**, starting from raw dataset processing to building an **interactive Power BI dashboard** for business insights.

The objective is to help financial institutions understand **risk patterns, customer segments, and financial activity** using data analytics.


# 🎯 Project Objectives

* Analyze banking customer data and financial transactions.
* Identify **loan distribution across income segments**.
* Understand **deposit and savings behavior**.
* Evaluate **credit card usage patterns**.
* Build an **interactive dashboard for business insights**.

# 📈 Key Insights from Analysis

* **Mid-income customers take the highest bank loans** compared to other income groups.
* **European customers have higher banking activity** compared to other regions.
* **Credit card balances are highest in the mid-income segment**.
* **Customer distribution by gender is nearly equal**, indicating balanced participation.
